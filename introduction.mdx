## Disbursement

APIs for mobile money and bank payouts

<CardGroup cols={2}>
  <Card title="MPESA B2C" icon="sparkles">
    This API allows a merchant to transfer funds to a customer's mobile wallet
  </Card>

  <Card title="Card" icon="sparkles">
    This API allows a merchant to transfer funds to a customer's bank account via Pesalink channel
  </Card>

  <Card title="MPESA B2B Paybill" icon="sparkles">
    This API allows a merchant to transfer funds to a business MPESA paybill
  </Card>

  <Card title="MPESA B2B Till" icon="sparkles">
    This API allows a merchant to transfer funds to a business MPESA paybill
  </Card>
</CardGroup>

## **Collections**

APIs for mobile money and bank collections.

<CardGroup cols={2}>
  <Card title="Mpesa C2B Paybill" icon="palette" href="https://mintlify.com/docs/settings/global">
    This API allows the merchants to receive payments through Kiotapay from customers who purchase their goods and services.
  </Card>

  <Card title="Reference APIs" icon="code" href="https://mintlify.com/docs/api-playground/openapi">
    This API allows merchants to collect bank payments on Kiotapay from their customers made through the bank channel.
  </Card>

  <Card title="Mpesa C2B STK Push" icon="stars" href="https://mintlify.com/customers">
    This API triggers a prompt on customer's phone containing amount, recipient details, account number. Transaction is completed by a pin in-put.
  </Card>
</CardGroup>
