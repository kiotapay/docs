# Get Single Paybill Transaction API

This API retrieves the details of a specific Paybill transaction using the provided `requestId` and `companyId`.

---

## **Endpoint**

`GET /api/v1/paybill/single`

---

## **Base URL**

`https://api.kiotapay.co`

---

## **Query Parameters**

| Parameter    | Type     | Description                                      | Required |
|--------------|----------|--------------------------------------------------|----------|
| `companyId`  | `string` | The ID of the company associated with the transaction. | ✅       |
| `requestId`  | `string` | The unique identifier for the specific transaction.   | ✅       |

---

## **Request Headers**

| Key            | Value              | Required |
|-----------------|--------------------|----------|
| `Content-Type` | `application/json` | ✅       |

---

## **Example Request**

### **cURL Command**

```bash
curl --location 'https://api.kiotapay.co/api/v1/paybill/single?companyId=4d6e1ea9-5ab8-496a-978f-7e9b2bce80c4&requestId=4f88dd2e-c4b4-4a3f-9605-1679686a6641'
