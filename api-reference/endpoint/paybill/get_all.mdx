# Get All Paybill Transactions API

This API retrieves all Paybill transactions for a specific company.

***

## **Endpoint**

`GET /api/sandbox/v1/paybill/all`

***

## **Base URL**

[`https://api.kiotapay.co`](https://api.kiotapay.co)

***


## **Request Headers**

| Key             | Value                 | Required | Description                                                        |
| --------------- | --------------------- | -------- | ------------------------------------------------------------------ |
| `Content-Type`  | `application/json`    | ✅        | Specifies that the request body is in JSON format.                 |
| `Authorization` | `Bearer <your-token>` | ✅        | API authentication token. Replace `<your-token>` with a valid JWT. |

***

## **Example Request**

### **cURL Command**

```bash
curl --location 'https://api.kiotapay.co/api/sandbox/v1/paybill/all'
```
