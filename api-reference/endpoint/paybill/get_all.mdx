# Get All Paybill Transactions API

This API retrieves all Paybill transactions for a specific company.

---

## **Endpoint**

`GET /api/v1/paybill/all`

---

## **Base URL**

`https://api.kiotapay.co`

---

## **Query Parameters**

| Parameter    | Type     | Description                                      | Required |
|--------------|----------|--------------------------------------------------|----------|
| `companyId`  | `string` | The ID of the company whose transactions are being queried. | ✅       |

---

## **Request Headers**

| Key             | Value                  | Required | Description |
|----------------|------------------------|----------|-------------|
| `Content-Type` | `application/json`      | ✅       | Specifies that the request body is in JSON format. |
| `Authorization` | `Bearer <your-token>`  | ✅       | API authentication token. Replace `<your-token>` with a valid JWT. |


---

## **Example Request**

### **cURL Command**

```bash
curl --location 'https://api.kiotapay.co/api/v1/paybill/all?companyId=4d6e1ea9-5ab8-496a-978f-7e9b2bce80c4'
