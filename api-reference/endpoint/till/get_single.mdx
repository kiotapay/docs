# Single Till Transaction API

This API retrieves the details of a specific till transaction based on the `companyId` and `requestId` provided.

## Endpoint

GET /api/v1/till/single

### Base URL

[`https://api.kiotapay.co`](https://api.kiotapay.co)

## Query Parameters

| Parameter   | Type     | Description                                            | Required |
| ----------- | -------- | ------------------------------------------------------ | -------- |
| `companyId` | `string` | The ID of the company initiating the request.          | ✅        |
| `requestId` | `string` | The unique ID of the request for the till transaction. | ✅        |

## **Request Headers**

| Key             | Value                 | Required | Description                                                        |
| --------------- | --------------------- | -------- | ------------------------------------------------------------------ |
| `Content-Type`  | `application/json`    | ✅        | Specifies that the request body is in JSON format.                 |
| `Authorization` | `Bearer <your-token>` | ✅        | API authentication token. Replace `<your-token>` with a valid JWT. |
