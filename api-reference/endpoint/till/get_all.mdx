# Get All Till Payments API

This API retrieves the list of all till payments for the given `companyId`.

## Endpoint

GET /api/v1/till/all


### Base URL

`https://api.kiotapay.co`

## Query Parameters

| Parameter   | Type     | Description                         | Required |
|-------------|----------|-------------------------------------|----------|
