# Till Status API

This API allows a company to fetch the status of a specific till transaction using the `companyId` and `requestId`.

## Endpoint

GET /api/v1/till/status


### Base URL

`https://api.kiotapay.co`

## Query Parameters

| Parameter   | Type     | Description                          | Required |
|-------------|----------|--------------------------------------|----------|
| `companyId` | `string` | The ID of the company initiating the request. | ✅       |
| `requestId` | `string` | The unique ID of the request for which the status is being checked. | ✅       |

## **Request Headers**

| Key             | Value                  | Required | Description |
|----------------|------------------------|----------|-------------|
| `Content-Type` | `application/json`      | ✅       | Specifies that the request body is in JSON format. |
| `Authorization` | `Bearer <your-token>`  | ✅       | API authentication token. Replace `<your-token>` with a valid JWT. |
