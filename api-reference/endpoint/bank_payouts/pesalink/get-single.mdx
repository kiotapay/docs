# Get Single Bank  Payment API

This API retrieves the details of a single Bank payment for an org by id

---

## **Endpoint**

`GET `/api/sandbox/v1/bank/payout/single

---

## **Base URL**

`https://api.kiotapay.co`

---

## **Query Parameters**

| Parameter   | Type     | Description                                         | Required |
| :---------- | :------- | :-------------------------------------------------- | :------- |
|             |          |                                                     |          |
| `requestId` | `string` | The unique identifier for the specific transaction. | ✅        |

---

## **Request Headers**

| Key             | Value                 | Required | Description                                                        |
| :-------------- | :-------------------- | :------- | :----------------------------------------------------------------- |
| `Content-Type`  | `application/json`    | ✅        | Specifies that the request body is in JSON format.                 |
| `Authorization` | `Bearer <your-token>` | ✅        | API authentication token. Replace `<your-token>` with a valid JWT. |

---

## **Example Request**

### **cURL Command**

```bash
curl --location 'https://api.kiotapay.co/api/sandbox/v1/bank/payout/single?
requestId=98b66546-0a83-4209-b037-adf5fb78d5ca' \
--header 'accept: application/json' \
--header 'Authorization: Bearer token'
```
