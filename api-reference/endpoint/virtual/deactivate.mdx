---
title: "Deactivate Sub-Accounts"
description: "This endpoint deactivates multi account subscription but does not close the already created relationship between a parent and all created sub-accounts."
---

## Endpoint

PUT /api/sandbox/v1/multiaccount/de-activate

### Base URL

`https://api.kiotapay.co`

## **Request Headers**

| Key             | Value                 | Required | Description                                                        |
| :-------------- | :-------------------- | :------- | :----------------------------------------------------------------- |
| `Content-Type`  | `application/json`    | ✅        | Specifies that the request body is in JSON format.                 |
| `Authorization` | `Bearer <your-token>` | ✅        | API authentication token. Replace `<your-token>` with a valid JWT. |

## 

**Sample Response Body**

```json
{
    "status": 200,
    "message": "Updated Successfully",
    "data": "un-subscribed Successfully"
}
```
