# Get All Send Money Transactions API

This API retrieves all send money transactions for a given `companyId`.

## Endpoint

GET /api/v1/send-money/all


### Base URL

`https://api.kiotapay.co`

## Query Parameters

| Parameter   | Type     | Description                         | Required |
|-------------|----------|-------------------------------------|----------|
| `companyId` | `string` | The ID of the company whose send money transactions are to be fetched. | âœ…       |
